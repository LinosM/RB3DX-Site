Bug Reports

We handle bug reports and track issues through GitHub, if you report bugs in our Discord or tell us directly, we will most likely miss it. The best way to let us know about a problem is to create an issue on the GitHub repository.

The current known issues can be found here. If you have a GitHub account, this is where you create your issue following the template.
